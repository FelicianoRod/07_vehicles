# 07_vehicles

# Configuración del entorno virtual
1. Creación del entorno virtual
```
conda create --name vehicles_env
```

2. Instalación de las librerías

```
conda install --name vehicles_env --file requirements.txt

```

3. Activar entorno virtual

```
conda activate vehicles_env
```


# Descripción

Análisis sobre la venta de coches en Estados Unidos, ofrece visualizaciones de gráficas:
- Distribución del Odómetro
- Relación entre Odómetro y Precio


# Funcionalidad

Selecciona las gráficas que deseas visualizar con los checkbox y luego crealas con el botón "construir"
